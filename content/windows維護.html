windows維護

在 tinyc.7z 中加上 python3114_git_putty.7z, 若已經在 s.cycu.org 登錄 id_rsa.pub 作為Github 公鑰, 則取下的對應 id_rsa 可以利用 puttygen 轉為 .ppk 使用

利用 putty 建立名稱與連線 session 都選為 github.com 的設定, 讓 Windows 與 s.cycu.org 進行 git clone 有相同的指令

開啟 puttygent, load 進先前從 s.cycu.org 主機 .ssh 目錄取下的 id_rsa, 將原先 OpenSSL 格式的私鑰存為 .ppk 格式

在 putty 所建立的 github.com session 必須指定位於 y: 目錄, 以免之後若隨身碟代號改變而失效

github.com session 設定 proxy 與 private key 位置後, 必須利用 Open 測試是否可以 ssh 連線到 github.com

putty session 設定完成後, 必須將位於 Computer\HKEY_CURRENT_USER\SOFTWARE\SimonTatham 的 .reg 導出, 之後只要利用滑鼠雙點擊就可以完成 putty session 設定

以 git clone --recurse-submodules git@github.com:帳號/倉儲名稱.git 取下倉儲內容, 進入倉儲後, 以 cms 啟動編輯網站, 改版後 Convert 然後利用 acp 推送到 Github
